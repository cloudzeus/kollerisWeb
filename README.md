# kollerisWeb

Website for [Kolleris](https://kolleris.com) — industrial & professional tools distributor since 1978.

Built with Next.js 16, next-intl (Greek/English), Tailwind CSS 4, and GSAP. Product, brand, and category data is pulled live from the hdctool.wwa.gr API.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Requires a `.env.local` with `HDCTOOL_BASE_URL`, `HDCTOOL_EMAIL`, `HDCTOOL_PASSWORD`.
