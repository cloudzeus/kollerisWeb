# KOLLERIS WEBSITE - DESIGN DIRECTIONS FOR DESIGNER

---

## 🎨 DESIGN PHILOSOPHY

**Core Approach:** Professional, trustworthy, action-oriented
**Not:** Flashy, trendy, overly complex
**Yes:** Clean, direct, confidence-building through specificity

The design should get out of the way and let the copy do the selling. Content is the hero, not the design.

---

## 📐 GENERAL LAYOUT PRINCIPLES

### Grid & Spacing
- **Grid:** 12-column responsive grid
- **Max width:** 1200px (desktop), full width (tablet), optimized mobile
- **Padding:** 20px mobile, 40px tablet, 60px desktop
- **Section spacing:** 60px (mobile), 80px (tablet), 100px (desktop) between major sections

### Typography
- **Headlines (H1):** 40px-56px, bold, conversational tone
- **Subheadings (H2):** 28px-32px, bold
- **Body text:** 16px, line-height 1.6, comfortable reading
- **Small text:** 14px max
- **All text should be readable without zooming**

### Color Palette
- **Primary:** Professional blue or dark (TBD - check any existing Kolleris branding)
- **Accent:** Orange or warm color for CTAs
- **Text:** Dark gray/charcoal for body, dark for headers
- **Background:** White or light gray
- **Borders:** Light gray dividers between sections

### Visual Hierarchy
Use size, color, and white space—not decoration—to guide attention.

## 🏠 HOMEPAGE STRUCTURE

### Section 1: Hero
**Height:** 60-70vh (tall and commanding)
**Layout:** 
- Large headline left/center (60% of space)
- Supporting subheading below
- Two CTA buttons (primary + secondary) stacked on mobile
- Optional: Background image/gradient (subtle, doesn't fight the text)

**Design Notes:**
- Headline should be immediately visible without scrolling
- Plenty of white space around text
- CTAs should be obvious (contrasting color, 16px+ font)

### Section 2: 4 Problem-Solution Cards
**Layout:** 2x2 grid (mobile: 1 column, stacked)
**Card style:**
- White or light gray background
- Border or subtle shadow (not both)
- Icon top-left or top-center (40px square)
- Bold problem text (red or dark)
- Regular solution text below
- Padding: 30px
- No background images—keep it clean

### Section 3: Featured Brands Carousel
**Layout:** Horizontal scrollable on mobile, grid on desktop
**Desktop:** 4-6 brands visible at once
**Mobile:** 2-3 brands visible, swipeable
**Each brand card:**
- Brand logo (100x100px max)
- Brand name below
- Hover: Subtle lift or border highlight
- Background: White or very light
- Spacing between items: 20px

**Note:** This loads from API—placeholder brands should show while loading

### Section 4: Featured Products Grid
**Layout:** 
- Desktop: 4 columns
- Tablet: 3 columns  
- Mobile: 2 columns, stacked

**Each product card:**
- Product image (square, 1:1 ratio)
- Product name (bold, 16px)
- Brand name (small, gray)
- Brief description (2 lines max, 14px)
- Stock status badge (green/red, top-right corner)
- "View Details" link (subtle, underline on hover)
- Spacing between cards: 20px

### Section 5: Services Overview
**Layout:** 3 columns (mobile: 1 column stacked)
**Each service block:**
- Icon (48px square, accent color)
- Heading (18px bold)
- Description (2-3 lines)
- Subtle border bottom
- No background color—just spacing

### Section 6: Bottom CTA
**Height:** 30vh
**Layout:** Centered, full width
**Elements:**
- Large question/headline (36px+)
- 2 buttons below (stacked on mobile)
- Subtle background color difference (off-white or very light gray)
- Generous padding (80px+ vertical)

### Footer
- Company name + year founded (left)
- Contact info (center or right)
- Social links (right or bottom)
- Copyright (bottom)
- Simple, no flourishes

## RESPONSIVE DESIGN SPECS

### Breakpoints
- **Mobile:** 0-640px
- **Tablet:** 641-1024px
- **Desktop:** 1025px+

### Mobile-First Principles
- Start with mobile layout
- Stack everything vertically
- 1 column for content
- CTAs full width or large touch targets (48px minimum)
- Readable text without zooming

### Tablet
- 2 columns where appropriate
- Larger text (readable from 12+ inches)
- Optimized grid layouts

### Desktop
- Full-featured layout
- Multi-column grids
- Hover states on interactive elements
- Optimized for 24" monitor viewing

## COMPONENT SPECS

### Buttons
- **Primary CTA:** Accent color, white text, 16px font, 12px padding vertical, 24px horizontal
- **Secondary CTA:** Outline/border style, accent color text, white background
- **Hover state:** Slightly darker or lift effect
- **Mobile:** Full width or larger

### Cards
- **Border:** 1px light gray OR subtle shadow (not both)
- **Padding:** 24px minimum
- **Border radius:** 4-8px (subtle, not rounded)
- **Hover:** Slight lift (2-4px shadow) on desktop

### Forms
- **Input fields:** 1px light border, 12px padding, 16px font
- **Focus state:** Accent color border, subtle background highlight
- **Labels:** Above fields, bold, 14px
- **Error messages:** Red text, 12px
- **Submit button:** Same as primary CTA

### Sectioning
- **Section padding:** 60px vertical (mobile), 80px (tablet), 100px (desktop)
- **Section backgrounds:** Alternate white and light gray for visual breathing room
- **Dividers between sections:** Optional, subtle light gray only

## IMAGES & MEDIA

### Product Images
- **Size:** 1:1 square ratio (200px - 400px depending on context)
- **Background:** White or transparent
- **Style:** Product on white background, no lifestyle shots
- **API integration:** Load from SoftOne

### Brand Logos
- **Size:** 100px - 150px
- **Format:** PNG with transparency preferred
- **Placement:** Centered in card
- **API integration:** Load from SoftOne

### Feature Icons
- **Size:** 40px - 80px depending on context
- **Style:** Line icons or solid icons (pick a style, be consistent)
- **Color:** Accent color
- **Style source:** Consider Lucide Icons or similar free sets

### Optional: Hero Images
- **Style:** Subtle, professional, not busy
- **Opacity:** Consider 30-40% opacity overlay so text is readable
- **Size:** High quality, optimized for web (no >200KB)
- **Content:** Could feature: warehouse, tools being used, team, shipping, manufacturing setting

### No Stock Photos
- Generic stock photos cheapen the site
- Either use real Kolleris photos or skip images
- Clean white space is better than bad stock photos

## INTERACTION DESIGN

### Hover States
- **Links:** Underline on hover
- **Buttons:** Color shift or lift effect
- **Cards:** Subtle shadow or border highlight
- **Don't:** Change layout or add distracting animations

### Micro-interactions
- **Form validation:** Show checkmark when field is valid
- **Loading states:** Show skeleton screens while API content loads
- **Transitions:** Smooth 0.2-0.3s ease-in-out
- **Scroll:** No parallax or fancy scroll effects (keep it professional)

### Accessibility
- **Color contrast:** 4.5:1 minimum for all text
- **Touch targets:** 48px minimum on mobile
- **Focus states:** Visible outline on all interactive elements
- **Alt text:** Required for all images
- **Semantic HTML:** Use proper heading hierarchy

## LANGUAGE SELECTOR

**Placement:** Top right corner
**Style:** Simple dropdown or flag icons (EN / ΕΛ)
**Behavior:** 
- Should remember user preference (localStorage)
- Should load Greek or English content/URLs
- No flash of wrong language on page load

---

## LOADING STATES

**For API content (brands, products):**
- Show skeleton screens (gray placeholder boxes)
- Same layout as final content
- Animate gentle pulse effect
- Don't show spinning loaders

---

## PERFORMANCE NOTES FOR DESIGNER

### Images
- Optimize all images (WebP format with fallbacks)
- Use responsive images (srcset for different sizes)
- Lazy load images below the fold

### Code
- Minimize animations (smooth but brief)
- Defer non-critical JavaScript
- Use CSS animations instead of JavaScript where possible

### Font Loading
- Use system fonts or 1-2 web fonts maximum
- Ensure text is visible while fonts load (font-display: swap)

## SUMMARY FOR DESIGNER

**Think:** Professional B2B industrial site
**Tone:** Trustworthy, expert, action-oriented
**Approach:** Content-first design, clean layouts, confidence-building through specificity
**Color:** Professional palette (suggest: dark blue + orange accent)

**Components:** Cards, grids, accordions, forms
**Responsive:** Mobile-first, tested at 640px, 1024px, 1200px+
**No:** Skeuomorphism, unnecessary animations, stock photos, design flourishes
